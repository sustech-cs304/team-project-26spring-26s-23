# 给使用者

- 这页给谁看：第一次进入文档、想先按任务而不是按技术分层理解 CanDue 的普通使用者。
- 这页解决什么问题：帮你按“是什么、怎么开始、怎么配置、怎么聊天、有什么边界”的顺序找到对应页面。
- 当前覆盖到哪：覆盖桌面宿主上的助手工作台主线，包括认识产品定位、完成首次使用、配置模型、理解 thinking，以及判断当前限制。
- 当前状态：主阅读链已可用；部分能力主题仍有部分接通边界。

这条路径只讲你会直接操作到的动作、看到的结果，以及怎样判断当前能不能用。不需要先读系统架构，也不需要先理解前后端分层。

## 建议怎么读

### 先判断值不值得继续试

按这个顺序：

1. [CanDue 是什么](./what-is-candue.md)
2. [当前限制与能力边界](./limits.md)
3. 再决定要不要继续看上手和配置页。

### 你已经准备动手试

按这个顺序：

1. [5 分钟上手](./quickstart.md)
2. [模型与 Provider 设置](./model-setup.md)
3. [聊天与 Thinking](./chat-and-thinking.md)
4. 如果中途想判断边界，再看[当前限制与能力边界](./limits.md)。
